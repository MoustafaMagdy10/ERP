# 🧾Inventory ERP System (In Progress)

A modular ERP system developed using **Java**, **Spring Boot**, and **MySQL**, focusing on inventory and user management functionalities.

---

## 📌 Features

- ✅ User authentication & role-based access control (Spring Security)
- 📦 Product, User, and Category CRUD
- 📊 Invoices & Suppliers (in progress)
- 🔄 REST APIs with layered architecture
- 🔍 Database integration (MySQL)


---
## 🚧 Project Status

This project is currently under active development. Implemented features include:

- **User Authentication & Authorization**: Implemented using Spring Security with role-based access control (RBAC).
    
- **Product Management**: CRUD operations for managing product details.
    
- **User Management**: CRUD operations for managing user accounts.


#### Planned features:

- **Invoice Handling**: Creation and management of invoices.
    
- **Supplier Management**: Tracking supplier information and interactions.
    
- **Dynamic Reporting**: Generating reports based on various criteria.
    
---

## 🧩 Project Structure

The project follows a layered architecture:

- **Controller Layer**: Handles HTTP requests and responses.
    
- **Service Layer**: Contains business logic.
    
- **Repository Layer**: Interacts with the MySQL database.
